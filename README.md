# frame

#### 介绍

分布式后台常用技术集合。通过maven依赖简单的引入使用各种常用技术。最终达成大数据处理，istio的paas微服务技术架构

#### 使用方式

通过maven依赖的方式引用到项目模块即可。一般都要引入common，其他模块根据系统需要引入。
